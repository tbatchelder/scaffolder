# Contributing to Scaffolder

Thanks for your interest in contributing to Scaffolder!  
We welcome improvements, bug fixes, new ideas, and expansions to the agentic prompting knowledge base.

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

Please keep PRs focused and well documented.
